Instrucciones proyecto

Este proyecto es un sistema tipo Trello. Permite crear tableros, listas y tareas (asignar a usuarios, poner fechas límite, prioridades y etiquetar). Las tareas se pueden mover entre listas mediante drag & drop.

🧩 Tecnologías utilizadas

  • Backend: Django + Django REST Framework
  • Frontend: React + dnd-kit
  • Base de datos: SQLite
